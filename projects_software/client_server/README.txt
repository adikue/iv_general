Cleint server apllications.
Library 'client_server.h' is used. For more information look into sources or contact with Ivan Vovk (adikue@gmail.com)

Limitations:
	Now connection between this applications established via port 3490 and loopback interface(aka 'localhost')

For compile:
	#make rebuild

Usage:
	1. start server:
		#./app_server
	2. start client:
		#./app_client
	3. Communicate:
		type in client shell any command :)
	4. Look into logs on server :)
